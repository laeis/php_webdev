Для того чтобы приложение начало работать необходимо:
1. Заполнить данные в файле congig.php
    a. Указать хост базы данных(БД)                      // define( 'DB_HOST', 'localhost' );
    б. Указать имя пользователя для БД               // define( 'DB_USER', 'root' );
    в. Указать пароль пользователя для доступа к БД // define( 'DB_PASSWORD', '' );
    г. Указать имя таблицы БД                          // define( 'DB_NAME', 'test_webdev_base' );


!!! Внимание в зависимости от операционной системы на которой будет запущено приложение необходимо исправить константы UPLOAD_DIR и IS_WINDOWS в файле config.php

В папке /backup_test_db находится дамп базы данных с небольшим количеством демо данных

В папке /lib -используемые библиотеки.

В папке /app -файлы приложения.

Основная страница сайта /

Для хранения файла резюме используется поле типа (BLOB) в таблице и файл в папке UPLOAD, в приложении используются файлы, если необходимо можно реализовать функционал использования данных из БД.

Для получения данных в формате JSON необходимо обратится на URL /json , в классе реализован единственный метод который возвращает список резюме, при необходимости можно расширить и дополнить проверками.

Для сортировки данных в целях упрощения был использован плагин "DataTables Table plug-in for jQuery"
https://datatables.net/, при необходимости возможна доработка и создание собственной пагинации, сортировки и поиска используя AJAX.

Важно.
Метод обновление количества отзывов пригоден только если приложение будет использоваться небольшим количество пользователей в противном случае необходимо изменить концепцию.
